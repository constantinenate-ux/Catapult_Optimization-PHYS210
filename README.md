# PhysProject-Catapult_Optimization

## Overview

I modelled and investigated parameters affecting a simplified catapult, and then performed a phase-space analysis to optimize those parameters.
I used Euler's method to model the trajectories of the projectiles, while accounting for dynamic wind resistance. 

## Key Features

- Modeled numerical solutions using Euler's Method
- Used a phase-space analysis to visualize optimized parameters
- Created clear graphics and visualizations
